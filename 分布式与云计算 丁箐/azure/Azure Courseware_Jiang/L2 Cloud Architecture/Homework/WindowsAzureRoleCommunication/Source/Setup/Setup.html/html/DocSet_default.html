System Requirements

You must have the following items to complete this lab:

  • IIS 7 (with ASP.NET, WCF HTTP Activation)
  • Microsoft .NET Framework 3.5 SP1
  • Microsoft Visual Studio 2008 SP1 (or above)
  • Windows Azure Tools for Microsoft Visual Studio (November 2009)

Installation

You must perform the following steps to prepare your computer for Windows Azure Worker Role Communication lab.

Note:
  To perform the setup steps you need administrator privileges.

For convenience, much of the code you will be managing along this hands-on lab is available as Visual Studio code snippets. The setup needed for this lab consist on checking the prerequisites of this lab and installing the code snippets. To do this:

  1. Run the SetupLab.cmd file located under the lab's Source\Setup folder.
  2. Follow the instructions to check the dependencies and install any that is missing.
  3. Once all the prerequisites are correctly installed, follow the wizard instructions to install the code snippets.

The following are the steps to set up the Visual Studio snippets.

  1. When prompted, choose all items in the installer.

    Figure 1

  2. A dialog window appears warning that the file is not signed; choose Yes to proceed anyway.

  3. When prompted for the location of the snippets, highlight all snippets, click the check box next to My Code Snippets, and click Next.

  4. Click Next to continue.

    Figure 4

  5. Click Finish.
  6. The installation should proceed and install all snippets.
